# 49 Keyboard 부품 리스트

49키 핸드와이어드 + 무선(BLE) + USB-C 키보드 빌드.

## 확정 사양

| 항목 | 값 |
|------|---|
| 레이아웃 | 49키 (`docs/keyboard-plate.jscad` 기반) |
| 컨트롤러 | LOLIN S3 Mini (ESP32-S3FH4R2) |
| 펌웨어 | RMK (Rust, Vial 네이티브) |
| 키맵 편집기 | Vial (vial.rocks) |
| 연결 | USB-C 유선 + BLE 5.0 무선 자동 전환 |
| 전원 | 3×AA 알카라인 교체형 |
| 배터리 구획 | 케이스 일체형 슬롯 + 스프링 접점 |
| 커버 | 슬라이드 + 스냅핏 |
| 매트릭스 | 7×7 + diode per key |
| 플레이트 | 파라메트릭 (STL 출력 예정) |
| 케이스 | JSCAD 커스텀 3D 프린팅 |

## 의사결정 로그

| 결정 | 사유 |
|------|------|
| 펌웨어 RMK | QMK+BT+VIA 조합 불가. RMK = Vial(VIA 호환) 네이티브 + BLE/USB 듀얼 |
| MCU ESP32-S3 | nRF52840 대비 국내 구매 용이, USB-C 네이티브, RMK 공식 지원 |
| LOLIN S3 Mini | 27 GPIO, 공식 제품, USB-C |
| 3×AA 알카라인 교체형 | 리튬 수명(~2년) 대비 반영구. 장기 미사용 시 분리 가능 |
| 통합 슬롯 + 스프링 접점 | 완제품 홀더 대비 슬림. AA 표준 치수로 사전 설계 가능 |
| 슬라이드 + 스냅핏 커버 | 공구 없이 교체, 외부 돌출 없음 |
| UEW 0.6mm 단일 | 국내 구매 가능한 굵기. Row/Col 통일 단순화 |
| TPS63020 Buck-Boost | AA 전압 변동(4.5→2.7V) 전체에서 3.3V 유지 |
| 슬라이드 스위치 내부 배치 | 외부 돌출 없음. 배터리 커버 열어 조작 |
| 유연납 0.6mm | 작업성 우위. 환기 + 손 세정으로 안전 관리 |
| 캡톤 + 폼테이프 | 열수축튜브·순간접착제 대체. 가열 도구 불필요 |
| 만능보드 7×9cm | 부품 마운트 전용 (매트릭스는 직접 배선) |
